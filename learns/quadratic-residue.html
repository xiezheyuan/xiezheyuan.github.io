Euler 准则

定理（Euler's Criterion）：对于奇素数 ppp 和一个整数 nnn，有：

(np)≡n(p−1)/2(modp)\left(\frac{n}{p}\right)\equiv n^{(p-1)/2}\pmod{p} (pn​)≡n(p−1)/2(modp)

也就是说，若 n(p−1)/2≡1(modp)n^{(p-1)/2}\equiv 1\pmod{p}n(p−1)/2≡1(modp)，那么 nnn 是一个模 ppp 意义下的二次剩余。

证明

若 n,pn,pn,p 不互质，则必有 p∣np\mid np∣n，进而 p∣n(p−1)/2p\mid n^{(p-1)/2}p∣n(p−1)/2。故 n(p−1)/2≡0(modp)n^{(p-1)/2}\equiv 0\pmod{p}n(p−1)/2≡0(modp) 符合 Euler 准则。故下面只考虑 gcd⁡(n,p)\gcd(n,p)gcd(n,p) 的情况。

首先根据费马小定理，有 np−1≡1(modp)n^{p-1}\equiv 1\pmod{p}np−1≡1(modp)，故 (n(p−1)/2)2≡1(modp)(n^{(p-1)/2})^2\equiv 1\pmod{p}(n(p−1)/2)2≡1(modp)。因此 n(p−1)/2≡±1(modp)n^{(p-1)/2} \equiv \pm 1\pmod{p}n(p−1)/2≡±1(modp)。

因此只需要证明 n(p−1)/2≡1(modp)n^{(p-1)/2}\equiv 1\pmod{p}n(p−1)/2≡1(modp) 是 nnn 为模 ppp 意义下的二次剩余的充要条件即可。

必要性：

若 nnn 为模 ppp 意义下的二次剩余，则任取一个 xxx 满足 x2≡n(modp)x^2\equiv n\pmod{p}x2≡n(modp)。代入并应用费马小定理，得到 n(p−1)/2≡xp−1≡1(modp)n^{(p-1)/2}\equiv x^{p-1}\equiv 1\pmod{p}n(p−1)/2≡xp−1≡1(modp)，证毕。

充分性：

根据原根存在定理，可取模 ppp 意义下的一原根 ggg，则一定存在 nnn 的以一个离散对数 kkk，即 gk≡n(modp)g^k\equiv n\pmod{p}gk≡n(modp)。

根据费马小定理，得到 (p−1)∣(k(p−1)2)(p-1)\mid \left(\frac{k(p - 1)}{2}\right)(p−1)∣(2k(p−1)​)。那么 k2∈Z\frac{k}{2}\in \mathbb{Z}2k​∈Z，也就是 kkk 为偶数。

那么取 x=gk/2x=g^{k/2}x=gk/2，则可得到 x2≡n(modp)x^2\equiv n\pmod{p}x2≡n(modp)。故 nnn 是模 ppp 意义下的一个二次剩余，证毕。

Cipolla 算法

Cipolla 算法用于找到同余方程 x2≡n(modp)x^2\equiv n\pmod{p}x2≡n(modp) 的一组解，其中 ppp 为奇素数。

先利用 Euler 准则判定是否存在解。故下面只讨论有解的情况。

我们任意选择一个 aaa，使得 a2−na^2-na2−n 非二次剩余（可以用 Euler 准则判定）。

那么令 z2≡a2−n(modp)z^2\equiv a^2-n\pmod{p}z2≡a2−n(modp)。那么有 (a+z)p+1≡n(modp)(a+z)^{p+1}\equiv n\pmod{p}(a+z)p+1≡n(modp)（为什么呢？详见下文【正确性证明】），故 (a+z)(p+1)/2(a+z)^{(p+1)/2}(a+z)(p+1)/2 是一组解。

注意到一定是不存在整数 zzz 满足条件。不妨扩充数域（可以类比复数域），定义这个数域中的数 xxx 形如 a+bza+bza+bz，其中 a,b∈Z/pZa,b\in \mathbb{Z}/p\mathbb{Z}a,b∈Z/pZ。并且类比复数定义乘法即可。

于是你发现这个解虚部总是 000，于是便得到了一个解。

容易发现只需要证明 (a+z)p+1≡n(modp)(a+z)^{p+1}\equiv n\pmod{p}(a+z)p+1≡n(modp)，即可证明得到的解虚部总为 000（因为 nnn 是模 ppp 意义下的二次剩余），并且是一组合法的解。

时间复杂度证明

在算法流程中有一行：

我们任意选择一个 aaa，使得 a2−na^2-na2−n 非二次剩余（可以用 Euler 准则判定）。

那么需要证明 a2−na^2-na2−n 非二次剩余的概率。

首先给出一个引理：

对于奇素数 ppp，模 ppp 意义下的二次剩余共有 p−12\frac{p-1}{2}2p−1​ 个。

证明非常简单。由于形如 x2≡n(modp)x^2\equiv n\pmod{p}x2≡n(modp) 的解有 222 个，并且对于不同的 nnn（0<n<p0<n<p0<n<p），不会有公共解。所以只需要取遍 0<x<p0<x<p0<x<p，每两个取出的数会构成一个有解同余方程的所有解，所以一共有 p−12\frac{p-1}{2}2p−1​ 个有解方程，证毕。

那么任意取一个数 xxx，它为模 ppp 意义下的（非）二次剩余的概率为 12+O(1)\frac{1}{2}+O(1)21​+O(1)。

因此，可以近似看成每一次选取 aaa，a2−na^2-na2−n 非二次剩余的概率为 12+O(1)\frac{1}{2}+O(1)21​+O(1)，因此期望 222 次就可以找到合适的 aaa。

所以时间复杂度为 O(log⁡p)O(\log p)O(logp)，瓶颈在快速幂。

代码实现

下面给出模板题的实现：

#include <bits/stdc++.h>
using namespace std;

struct cipolla_complex {
    int x, y, us, mod;
    cipolla_complex(int _x, int _y, int _us, int _mod): x(_x), y(_y), us(_us), mod(_mod) {}
    cipolla_complex operator*(const cipolla_complex &rhs){
        return cipolla_complex(
            (1ll * x * rhs.x + (1ll * y * rhs.y % mod) * us) % mod,
            (1ll * x * rhs.y + 1ll * y * rhs.x) % mod,
            us, mod
        );
    }
};

cipolla_complex fastpow(cipolla_complex a, int n){
    cipolla_complex res = {1, 0, a.us, a.mod};
    while(n){
        if(n & 1) res = res * a;
        a = a * a, n >>= 1;
    }
    return res;
}

int fastpow(int x, int y, int mod){
    int ret = 1;
    while(y){
        if(y & 1) ret = 1ll * ret * x % mod;
        x = 1ll * x * x % mod, y >>= 1;
    }
    return ret;
}

mt19937 rnd(chrono::steady_clock::now().time_since_epoch().count());

int cipolla(int n, int mod){
    if(fastpow(n, (mod - 1) >> 1, mod) != 1) return -1;
    int a, us;
    do {
        a = rnd() % mod, us = ((1ll * a * a + mod) - n) % mod;
    } while(fastpow(us, (mod - 1) >> 1, mod) != mod - 1);
    return fastpow(cipolla_complex(a, 1, us, mod), (mod + 1) >> 1).x;
}

signed main(){
    ios::sync_with_stdio(false);
    cin.tie(0), cout.tie(0);
    int t; cin >> t;
    while(t--){
        int n, mod; cin >> n >> mod;
        if(n == 0) {cout << 0 << '\n'; continue; }
        int ans = cipolla(n, mod);
        if(!(~ans)) cout << "Hola!" << '\n';
        else cout << min(ans, mod - ans) << ' ' << max(ans, mod - ans) << '\n';
    }
    return 0;
}
