集合幂级数概论

集合幂级数

定义

设全集 U={1,2,⋯ ,n},U={S∣S⊆U}U=\{1,2,\cdots,n\},\mathbf{U}=\{S\mid S\subseteq U\}U={1,2,⋯,n},U={S∣S⊆U}。定义 UUU 上的集合幂级数 f:U→Rf:\mathbf{U}\to Rf:U→R，其中 RRR 为一交换环。令 fSf_SfS​ 表示 fff 上 SSS 的像。

若 f:U→R,g:U→Rf:\mathbf{U}\to R,g:\mathbf{U}\to Rf:U→R,g:U→R，则定义：

  • h=f+gh=f+gh=f+g，其中 h:U→Rh:\mathbf{U}\to Rh:U→R 且 hS=fS+gSh_S=f_S+g_ShS​=fS​+gS​。类似的可以定义 h=f−gh=f-gh=f−g。
  • h=kfh=kfh=kf，其中 h:U→R,k∈Rh:\mathbf{U}\to R,k\in Rh:U→R,k∈R，其中 hS=kfSh_S=kf_ShS​=kfS​，类似的可以定义 h=−fh=-fh=−f。
  • h=f×gh=f\times gh=f×g，其中 h:U→Rh:\mathbf{U}\to Rh:U→R 且 hS=∑T⊆SfTgS\Th_S=\sum\limits_{T\subseteq S} f_Tg_{S\backslash T}hS​=T⊆S∑​fT​gS\T​，即 f,gf,gf,g 的子集卷积（不交并卷积）。
  • h=f∗gh=f\ast gh=f∗g，其中 h:U→Rh:\mathbf{U}\to Rh:U→R 且 hS=∑A∪B=SfAgBh_S=\sum\limits_{A\cup B=S} f_Ag_BhS​=A∪B=S∑​fA​gB​，即 f,gf,gf,g 的集合并卷积（OR 卷积）。
  • h=f∘gh=f\circ gh=f∘g，其中 h:U→Rh:\mathbf{U}\to Rh:U→R 且 hS=fS⋅gSh_S=f_S\cdot g_ShS​=fS​⋅gS​，即 f,gf,gf,g 的逐项积。

另外还可以定义集合交卷积（AND 卷积）、集合对称差卷积（XOR 卷积）等，它们在一般的 FMT/FWT 技术中是十分重要的，不过它们在集合幂级数理论中应用较少，故不再赘述。

若令 U={1,2,⋯ ,n}U=\{1,2,\cdots,n\}U={1,2,⋯,n} 上的全体集合幂级数集合为 F(U,R)\mathbf{F}(U,R)F(U,R)，则可以验证代数结构 (F(U,R),+,×)(\mathbf{F}(U,R),+,\times)(F(U,R),+,×) 是一个交换环，并是一个线性空间（若 RRR 为数域）。

我们可以取线性空间的一组基，共有 2n2^n2n 个基向量。因此可以构建一个 UUU 的子集到基向量的映射 f:U→F(U,R)f:U\to \mathbf{F}(U,R)f:U→F(U,R) 其中 fS,T=[S=T]f_{S,T}=[S=T]fS,T​=[S=T]，也就是 one-hot 的集合幂级数，并将这样的集合幂级数 fSf_SfS​ 记作 zSz^SzS，其中 zzz 是占位元，不代表实际的数。

根据基的性质，我们可以将任意一个 F(U,R)\mathbf{F}(U,R)F(U,R) 中的集合幂级数表示为若干个基的数乘之和的形式，即：

f=∑S⊆UfSzSf = \sum_{S\subseteq U} f_S z^S f=S⊆U∑​fS​zS

仿照级数的，我们也可以定义集合幂级数的取系数算子 [zS]f=fS[z^S]f = f_S[zS]f=fS​，如果 RRR 是一个多项式环，还可以组合级数的取系数算子使用，如 [zSxk]f=[xk]fS[z^Sx^k]f=[x^k]f_S[zSxk]f=[xk]fS​。

这样我们就成功定义了集合幂级数以及其相关的运算（线性运算、卷积运算），并且让它具有类似一般地级数的形式。

快速莫比乌斯变换

快速莫比乌斯变换用于快速计算集合 OR 卷积。

定义

对于 F(U,R)\mathbf{F}(U,R)F(U,R) 中的集合幂级数 fff，定义其莫比乌斯变换（Moebius Transform） FMT(f)\textsf{FMT}(f)FMT(f) 为：

FMT(f)=∑S⊆UzS∑T⊆SfT\textsf{FMT}(f)=\sum_{S\subseteq U}z^S\sum_{T\subseteq S}f_T FMT(f)=S⊆U∑​zST⊆S∑​fT​

即 fff 的子集和。

定义 fff 的莫比乌斯逆变换 IFMT(f)\textsf{IFMT}(f)IFMT(f) 满足 FMT(IFMT)=f\textsf{FMT}(\textsf{IFMT})=fFMT(IFMT)=f。

我们将给出一个引理，证明莫比乌斯变换可以帮助我们计算 OR 卷积：

引理

对于 F(U,R)\mathbf{F}(U,R)F(U,R) 中的集合幂级数 f,gf,gf,g，有：

FMT(f∗g)=FMT(f)∘FMT(g)\textsf{FMT}(f\ast g) = \textsf{FMT}(f) \circ \textsf{FMT}(g) FMT(f∗g)=FMT(f)∘FMT(g)

证明这个引理也不难，只需要按照各部分定义变形即可：

FMT(f)∘FMT(g)=∑S⊆U(∑T⊆SfT)(∑T⊆SgT)zS=∑S⊆UzS∑A⊆S,B⊆SfAgB=∑S⊆UzS∑T⊆S∑A∪B=TfAgB=FMT(f∗g)\begin{aligned} \textsf{FMT}(f) \circ \textsf{FMT}(g)&=\sum_{S\subseteq U} \left(\sum_{T\subseteq S} f_T\right) \left(\sum_{T\subseteq S} g_T\right) z^S = \sum_{S\subseteq U} z^S \sum_{A\subseteq S, B\subseteq S} f_A g_B \\ &= \sum_{S\subseteq U} z^S \sum_{T\subseteq S} \sum_{A\cup B=T} f_A g_B = \textsf{FMT}(f\ast g) \end{aligned} FMT(f)∘FMT(g)​=S⊆U∑​(T⊆S∑​fT​)(T⊆S∑​gT​)zS=S⊆U∑​zSA⊆S,B⊆S∑​fA​gB​=S⊆U∑​zST⊆S∑​A∪B=T∑​fA​gB​=FMT(f∗g)​

根据该定理，我们得到了一个快速求 OR 卷积的方法：先将 f,gf,gf,g 进行莫比乌斯变换 FMT(f),FMT(g)\textsf{FMT}(f),\textsf{FMT}(g)FMT(f),FMT(g)，然后计算这两个集合幂级数的逐项乘积 h=f∘gh=f\circ gh=f∘g，最后 IFMT(h)\textsf{IFMT}(h)IFMT(h) 即为 f∗gf\ast gf∗g 的值。

那么现在我们需要快速进行莫比乌斯变换以及其逆变换。注意到这个莫比乌斯变换的形式和高维前缀和一模一样，莫比乌斯逆变换对应高维差分，因此可以使用高维前缀和的标准方法：Sum over Subsets DP 来做。

Sum over Subsets DP 求高维前缀和

直接给出一个代码吧，这个东西挺简单的：

void fmt(int *a, int n){
    for(int i=1;i<=n;i++){
        for(int j=0;j<(1<<n);j++){
            if((j >> (i - 1)) & 1) a[j] = Add(a[j], a[j ^ (1 << (i - 1))]);
        }
    }
}

void ifmt(int *a, int n){
    for(int i=1;i<=n;i++){
        for(int j=(1<<n)-1;~j;j--){
            if((j >> (i - 1)) & 1) a[j] = Sub(a[j], a[j ^ (1 << (i - 1))]);
        }
    }
}

由于 Sum over Subsets DP 可以用于快速进行莫比乌斯变换，所以又称为快速莫比乌斯变换（Fast Moebius Transform, FMT）。

另外莫比乌斯变换还有一个重要性质：

引理

对于 F(U,R)\mathbf{F}(U,R)F(U,R) 中的集合幂级数 f,gf,gf,g，有：

FMT(f+g)=FMT(f)+FMT(g)∀k∈R,kFMT(f)=FMT(kf)\begin{aligned} &\textsf{FMT}(f + g) = \textsf{FMT}(f) + \textsf{FMT}(g)\\ &\forall k\in R, k\textsf{FMT}(f) = \textsf{FMT}(kf) \end{aligned} ​FMT(f+g)=FMT(f)+FMT(g)∀k∈R,kFMT(f)=FMT(kf)​

即莫比乌斯变换是线性的。

这两条性质都是显然的，其原因是 RRR 中的乘法分配律与加法交换律。

OR 卷积的参考实现。

占位多项式与子集卷积

首先我们需要知道一个经典推论：

推论

zS×zT={zS∪TS∩T=∅0otherwisez^S\times z^T=\begin{cases} z^{S\cup T}& S\cap T=\varnothing\\ 0& \text{otherwise} \end{cases} zS×zT={zS∪T0​S∩T=∅otherwise​

这个推论可以清晰的描述集合幂级数的子集卷积。

考虑集合的不交并运算，S⊔TS \sqcup TS⊔T 可以看成 S∪TS\cup TS∪T，不过要求 ∣S∪T∣=∣S∣+∣T∣|S\cup T|=|S|+|T|∣S∪T∣=∣S∣+∣T∣。因此可以考虑在集合幂级数中体现集合的大小信息。而集合大小是加性的，非常适合用形式幂级数描述。

定义

F(U,R)\mathbf{F}(U,R)F(U,R) 中的集合幂级数 fff 的占位多项式为 f‾∈F(U,R[x])\overline{f}\in\mathbf{F}(U,R[x])f​∈F(U,R[x])。其中 R[x]R[x]R[x] 表示 RRR 为系数，xxx 为占位元的多项式环，同时 f‾S=fSx∣S∣\overline{f}_S = f_S x^{|S|}f​S​=fS​x∣S∣。

然后我们将证明一个定理：

定理

对于 F(U,R)\mathbf{F}(U,R)F(U,R) 中的集合幂级数 f,gf,gf,g，有：

(f×g)S=[zSx∣S∣](f‾∗g‾)(f\times g)_S = [z^S x^{|S|}](\overline{f} \ast \overline{g}) (f×g)S​=[zSx∣S∣](f​∗g​)

证明如下：

f‾∗g‾=∑S⊆UzS∑A∪B=SfAgBx∣A∣+∣B∣  ⟹  [zSx∣S∣](f‾∗g‾)=∑S⊆U∑A∪B=S,A∩B=∅fAgB=∑S⊆U∑T⊆SfTgS\T=(f×g)S\begin{aligned} &\overline{f} \ast \overline{g}=\sum_{S\subseteq U} z^S \sum_{A\cup B=S} f_A g_B x^{|A|+|B|}\\ &\implies [z^S x^{|S|}](\overline{f} \ast \overline{g})=\sum_{S\subseteq U} \sum_{A\cup B=S, A\cap B=\varnothing} f_A g_B = \sum_{S\subseteq U}\sum_{T\subseteq S} f_T g_{S\backslash T} = (f\times g)_S \end{aligned} ​f​∗g​=S⊆U∑​zSA∪B=S∑​fA​gB​x∣A∣+∣B∣⟹[zSx∣S∣](f​∗g​)=S⊆U∑​A∪B=S,A∩B=∅∑​fA​gB​=S⊆U∑​T⊆S∑​fT​gS\T​=(f×g)S​​

因此我们只需要将占位多项式按照 OR 卷积的方法进行运算，最后的结果 hhh 中的 [zSx∣S∣]h[z^{S}x^{|S|}]h[zSx∣S∣]h 就是子集卷积中 SSS 对应的结果。

FMT / IFMT 操作只涉及到了元素的加法，多项式加法时间复杂度 O(n)O(n)O(n)。而计算逐项乘法则需要多项式卷积，这一步时间复杂度为 O(n2)O(n^2)O(n2) 或 O(nlog⁡n)O(n\log n)O(nlogn)（FFT / NTT），不过在集合幂级数计算中，nnn 往往较小（≤20\leq 20≤20），因此采用 O(n2)O(n^2)O(n2) 的暴力常数更好、跑的更快。

最终我们得到了一个 O(2nPolyMul(n))O(2^n \mathrm{PolyMul}(n))O(2nPolyMul(n)) 的算法，其中 PolyMul(n)\mathrm{PolyMul}(n)PolyMul(n) 一般取 O(n2)O(n^2)O(n2)。

定义

对于多项式 g(x)g(x)g(x) 与 F(U,R)\mathbf{F}(U,R)F(U,R) 中的集合幂级数 fff，其中 ∣U∣=n|U|=n∣U∣=n。f,gf,gf,g 的复合 g(f)g(f)g(f) 定义为：

g(f)=∑i=0n[xi]g(x)⋅fig(f)=\sum_{i=0}^{n} [x^i]g(x)\cdot f^i g(f)=i=0∑n​[xi]g(x)⋅fi

考虑如何计算复合。尝试取结果的莫比乌斯变换，然后应用 FMT 的线性性：

FMT(g(f))=∑i=0n[xi]g(x)⋅FMT(fi)\begin{aligned} \textsf{FMT}(g(f))&=\sum_{i=0}^{n} [x^i]g(x)\cdot \textsf{FMT}(f^i) \end{aligned} FMT(g(f))​=i=0∑n​[xi]g(x)⋅FMT(fi)​

又因为：

[zS]fi=[zSx∣S∣]f‾∗f‾∗⋯∗f‾⏟i=[zSx∣S∣]IFMT(FMT(f‾)∘FMT(f‾)∘⋯∘FMT(f‾)⏟i)  ⟹  [zS]FMT(fi)=[zSx∣S∣](FMT(f‾)∘FMT(f‾)∘⋯∘FMT(f‾)⏟i)\begin{aligned} &[z^S]f^i=[z^S x^{|S|}]\underbrace{\overline{f}\ast \overline{f}\ast \cdots\ast \overline{f}}_{i}=[z^S x^{|S|}]\textsf{IFMT}\left({\underbrace{\textsf{FMT}(\overline{f})\circ \textsf{FMT}(\overline{f})\circ \cdots\circ \textsf{FMT}(\overline{f})}_{i}}\right)\\ &\implies [z^S]\textsf{FMT}(f^i)=[z^S x^{|S|}]\left({\underbrace{\textsf{FMT}(\overline{f})\circ \textsf{FMT}(\overline{f})\circ \cdots\circ \textsf{FMT}(\overline{f})}_{i}}\right) \end{aligned} ​[zS]fi=[zSx∣S∣]if​∗f​∗⋯∗f​​​=[zSx∣S∣]IFMT​iFMT(f​)∘FMT(f​)∘⋯∘FMT(f​)​​​⟹[zS]FMT(fi)=[zSx∣S∣]​iFMT(f​)∘FMT(f​)∘⋯∘FMT(f​)​​​​

[zS]FMT(g(f))=[zSx∣S∣]∑i=0n[xi]g(x)⋅(FMT(f‾)∘FMT(f‾)∘⋯∘FMT(f‾)⏟i)=[x∣S∣]∑i=0n[xi]g(x)([zS]FMT(f‾))i=[x∣S∣]g([zS]FMT(f‾))\begin{aligned} [z^S]\textsf{FMT}(g(f))&=[z^S x^{|S|}]\sum_{i=0}^{n} [x^i]g(x)\cdot \left({\underbrace{\textsf{FMT}(\overline{f})\circ \textsf{FMT}(\overline{f})\circ \cdots\circ \textsf{FMT}(\overline{f})}_{i}}\right)\\ &=[x^{|S|}]\sum_{i=0}^{n}[x^i]g(x)([z^S] \textsf{FMT}(\overline{f}))^i\\ &=[x^{|S|}]g([z^S] \textsf{FMT}(\overline{f})) \end{aligned} [zS]FMT(g(f))​=[zSx∣S∣]i=0∑n​[xi]g(x)⋅​iFMT(f​)∘FMT(f​)∘⋯∘FMT(f​)​​​=[x∣S∣]i=0∑n​[xi]g(x)([zS]FMT(f​))i=[x∣S∣]g([zS]FMT(f​))​

集合幂级数的逆 / exp / ln

在上节我们注意到我们有 O(2nn3)O(2^nn^3)O(2nn3) 的一般集合幂级数多项式复合的算法，在这一节我们讨论特殊的集合幂级数复合问题。

对于一般的多项式，求逆、exp⁡,ln⁡\exp,\lnexp,ln 都有 O(n2)O(n^2)O(n2) 的朴素算法和 O(nlog⁡n)O(n\log n)O(nlogn) 的快速算法（基于 FFT / NTT 和牛顿迭代）。

由于求逆、exp⁡\expexp 和 ln⁡\lnln 都可以看成特殊的多项式复合，因此也可以采用上述的步骤，对集合幂级数求 exp⁡,ln⁡\exp, \lnexp,ln 和逆，时间复杂度为 O(2nn2)O(2^nn^2)O(2nn2)。

另外生成幂级数有所谓【exp⁡\expexp 的组合意义】，指的是有标号元素构成的集合来生成集族，那么集合幂级数也可以拥有 exp⁡\expexp 的组合意义，即将一个集合划分为多个子集，求每一个子集的权值的乘积之和。这也是很容易理解的。

Part 1

连通子图计数

给定一个 nnn 个点的无向图，求其连通子图的数量。1≤n≤201\leq n\leq 201≤n≤20。

考虑 exp⁡\expexp 的组合意义，任意子图是由若干个连通子图（连通块）组成的。若定义集合幂级数 f,gf,gf,g，fSf_SfS​ 为点集为 SSS 的连通子图数量，gSg_SgS​ 为点集为 SSS 的子图数量。那么有 g=exp⁡fg=\exp fg=expf，即 f=ln⁡gf=\ln gf=lng。

又因为 gSg_SgS​ 是容易求得的：

gS=2∑(u,v)∈G[u∈S∧v∈S]g_S=2^{\sum_{(u,v)\in G} [u\in S\land v\in S]} gS​=2∑(u,v)∈G​[u∈S∧v∈S]

所以 fff 可以用集合幂级数的 ln⁡\lnln 求。时间复杂度 O(2nn2)O(2^n n^2)O(2nn2)。
